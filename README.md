# Person & Animal Classifier

## Description
This project uses a Convolutional Neural Network (CNN) built with TensorFlow/Keras to classify images as either "Person" or "Animal" with 89% mAP50 accuracy.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- OpenCV (if you used it)

## How To Run
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Download the trained model from [GitHub Releases](link-to-your-release)
4. Run: `python predict.py --image path/to/image.jpg`

## Dataset
Dataset was taken from Flicker8k on github.

## Results
- Accuracy: 89% mAP50
